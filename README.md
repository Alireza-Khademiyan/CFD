Domain
$$
0  < X < 1
$$

Governing equation

$$
\frac{\partial^2 \phi}{\partial x^2} = S(x)
$$
where $$ S(x) = x(x-1) $$ 

Boundary conditions

$$
\phi(x = 0) = \phi(x = 1) = 0 
$$

Discritization (cintral in space)
$$
\frac{\partial^2 \phi}{\partial x^2} \Big|_i = S(x_i)
$$
where 
$$
\frac{\partial^2 \phi}{\partial x^2} \Big|_i = \frac{\phi(x)_{i-1} - 2\phi(x)_{i} + \phi(x)_{i+1}}{\Delta x ^2}
$$
and
$$
S(x_i) = x_i ( x_i - 1 )
$$
so
$$
\phi(x)_{i-1} -2\phi(x)_{i} + \phi(x)_{i+1}= \Delta x ^2 x_i(1 - x_i)
$$

```python
def Numerical(N):
    N = 10
    L = 1
    dx = L/(N-1)
    A = np.zeros((N,N))
    B = np.zeros((N,1))
    for i in range(N):
        xi = (i) * dx
    #For A
        if i == 0:
            A[i,i] = 1
        elif i == N-1:
            A[i,i] = 1
        else:
            A[i , i] = -2
            A[i , i-1] = A[i , i+1] = 1
    #For B
            B[i] = dx**2 * (xi) * (xi-1) 
    x = np.linalg.solve(A, B)
    return x
```


```python
def Analytical(N):
    phi_exact = np.zeros(N)
    for i in range(N):
        xi = i*dx
        phi_exact[i] = xi**4 / 12 - xi**3/6 + xi / 12
    return phi_exact
```


```python
n = 10
N = 10
Norms = np.zeros(n)
number = np.zeros(n)
i = 1
while i <= n:
    Num = Numerical(i * N)
    Ana = Analytical(i * N)
    number[i-1] = L/(i*N-1)

    Norms[i-1] = norm
```
